# .NET Developer Study Plan

Visual roadmap from beginner to job-ready .NET developer with Mermaid.js diagrams.

## Quick Start

```bash
# Install dependencies
npm install

# Start dev server
npm run dev
```

Then open `http://localhost:3001` in your browser.
